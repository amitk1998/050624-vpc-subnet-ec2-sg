# **Azure Application Gateway Module**

Terraform module to create Azure Application Gateway

# **Description**

This module is used to create an Azure Application Gateway. It requires several attributes in order to be created on Azure, including `name`, `location`, `resource_group_name`, etc.

## Example Usage

```hcl
module "application_gateway" {
  source = "terraform-azure-modules/application-gateway/azurerm"

  name                 = "my-app-gateway"
  resource_group_name  = "my-resource-group"
  location             = "East US"
  backend_address_pool = {
    name         = "backend-pool"
    fqdns        = ["example.com"]
    ip_addresses = ["10.0.0.1", "10.0.0.2"]
  }

  # Add more configuration here as needed
}
